Keyboard Shortcuts

Shortcut Guide

Global Navigation

  • F1: Go to Dashboard
  • F2: Go to Discover
  • F3: Go to Library
  • F4: Go to Calendar
  • Ctrl + ,: Open Settings
  • Esc: Go Back
  • F / F11: Toggle Fullscreen

Player Controls

  • Space: Pause/Play
  • ← / →: Seek Backward/Forward 10s
  • Shift + ← / →: Seek Backward/Forward 1m
  • ↑ / ↓: Volume Up/Down
  • M: Mute/Unmute
  • S: Toggle Subtitles
  • A: Open Audio Menu

Tab Navigation

  • Ctrl + Tab: Next Tab
  • Ctrl + Shift + Tab: Previous Tab
  • S: Focus Search

Other Features

  • D: Show Playback Stats
  • I: Show Video Info
  • P: Toggle Player Modes
  • Ctrl + Shift + I: Open Dev Tools
